# nano-motion-particles
How to Interact with the System
Once you open the interface in your browser and allow webcam access:

Gesture	Engine Action
Move Hand Around	The overall particle structure shifts and follows your hand dynamically across the screen space (X,Y coordinate tracking).
Push closer/pull away	Modifies the base color map (Z-depth mapping calculation updating properties inside the animation rendering matrix loop).
Open Palm	Forces particles to fly outwards, expanding the structure drastically.
Close Fist	Implodes the particles tightly toward your hand core, resulting in a condensed star formation cluster.
Pinch Finger and Thumb	Triggers the Template Morph Engine! Hold a distinct index-to-thumb pinch for a brief second to watch the system smoothly shift arrays through Galaxy → Heart → Flower → Saturn Rings → Fireworks Explosion.
